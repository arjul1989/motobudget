# Links de Configuración GCP

## 1. Activar Facturación (Requerido)
El proyecto **ManoMotos** ya está creado, pero está "congelado" esperando que le vincules una cuenta de facturación (tarjeta).

**[HAZ CLIC AQUÍ PARA ACTIVAR LA FACTURACIÓN](https://console.cloud.google.com/billing/linkedaccount?project=mano-motos-app)**

1.  Entra al link.
2.  Si ya tienes una cuenta de facturación, selecciónala.
3.  Si no, crea una nueva (te pedirán tarjeta, pero no te cobrarán si usas la capa gratuita).
